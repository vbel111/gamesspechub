2 GitHub Personal Access Token

Create a GitHub Personal Access Token to allow the blog system to publish posts:

  1. Go to GitHub Settings ‚Üí Developer Settings ‚Üí Personal Access Tokens
  2. Click "Generate new token (classic)"
  3. Give it a name like "GameSpecsHub Blog"
  4. Select these scopes:
    • repo (Full control of private repositories)
    • workflow (Update GitHub Action workflows)
  5. Click "Generate token"
  6. Copy the token (it starts with ghp_)

3 Image Upload Setup (Optional)

For image uploads, you can use ImgBB (free service):

  1. Go to imgbb.com
  2. Create an account
  3. Get your API key from your account settings
  4. Replace YOUR_IMGBB_API_KEY in admin.html

4 Update Configuration Files

Update the configuration in your files:

// In admin.html, update these lines: const owner = "YOUR_GITHUB_USERNAME"; const repo = "YOUR_REPOSITORY_NAME"; const imgbbAPI = "YOUR_IMGBB_API_KEY";
